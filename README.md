# Portfolio-Website für angehenden Systemintegrator

Dies ist der Quellcode meiner persönlichen Portfolio-Website, die ich mit HTML, CSS und JavaScript erstellt habe. Die Website ist darauf ausgelegt, potenziellen Arbeitgebern einen Überblick über meine Fähigkeiten, Projekte und Erfahrungen als angehender Systemintegrator zu geben.

## Features

- Modernes, minimalistisches Design mit dunklem Farbschema
- Vollständig responsiv (Mobile, Tablet, Desktop)
- Sanfte Animationen und Hover-Effekte
- Separate Unterseiten für Über mich, Fähigkeiten, Projekte, Erfahrung und Kontakt
- Integration von GitHub-Links und Badges
- SEO-optimiert mit Meta-Tags
- Deploybar auf GitHub Pages

## Verwendete Technologien

- HTML5
- CSS3 (Flexbox, Grid, Custom Properties)
- JavaScript (Intersection Observer, Smooth Scrolling)
- Google Fonts (Inter)
- Font Awesome Icons
